O que é HTML?

(voltar)

HTML significa "HyperText Markup Language" (Linguagem de Marcação de Hipertexto) e pode se definir como uma linguagem de marcação de texto, usada principalmente para a criação de websites. A criação do HTML surgiu em 1991 pelo físico britânico Tim Berners-Lee que inventou a World Wide Web.


Estrutura básica do HTML

(voltar)

O html possui uma estrutura básica que determina as Principais informações para a criação de uma página Web, como mostra o exemplo a seguir:

  • DOCTYPE: informa ao navegador que o documento é escrito em HTML5.
  • HTML: Representa todo o conteúdo da página.
  • HEAD: Contém informações como o título da página e metadados.
  • TITLE: título da página, que aparece na guia do navegador e em resultados de busca.
  • BODY: contém todo o conteúdo visível da página

  • Tag(u): Adiciona uma linha


Atributo das tags

(voltar)

As tags html podem utilizar de diversos atributos que definem suas características e a forma como serão utilizadas, alguns atributos mais genéricos são:

  • Id: Identifica o elemento.
  • Style: Define um estilo CSS.
  • Class: Ajuda a padronizar para o CSS.

Outros exemplos de atributos:

  • Input type: Serve para criar campos onde podemos definir o tipo, os atributos mais usados são o "number" e "text".
  • Img src: Usada para adicionar imagens, é necessário o atributo "SRC" para adicionar o endereço da imagem.
  • Width: usado para definir a largura de uma imagem ou elemento.
  • Href: Identifica o destino de um link.
  • Target: Especificar onde o link deve ser aberto.

Existem muitos outros atributos disponíveis, dependendo do tipo de tag e do objetivo específico.


Tipografia

(voltar)

A tipografia html é usada para definir as formatações do texto que é exibido na página web.

Existem inúmeras tags para formatar um texto porém as mais utilizdas são:

  • H1 a H6: Definem o título e o subtítitulo
  • Strong: Deixa o texto em negrito
  • P : Usada para adicionar um parágrafo
  • i : Deixa o texto em itálico


Diferença entre listas ordenadas e não ordenadas

(voltar)

As listas ordenadas do html podem ser divididas em dois tipos, as ordenadas e as não ordenadas.

As não ordenadas são representadas pelo "UL" e podem ser utilizadas para criar listas sem numeração ou hierarquia, ao invés de números nessas listas os itens são marcados por pontos mas podem ser estilizados pelo CSS.

Já as ordenadas são representadas pelo "OL" e são utilizadas para dar uma ordem espécifica aos itens, sendo assim possuem numeração porém também podem ser estilizadas.
